Remove untranslated menu items from menu by simply removing the language string.
Use when a menu should appear in one language only.

== Description ==

For example if you remove the French title for a page:
<!--:fr--><!--:--><!--:en-->Company Profile<!--:-->
Without our plugin, you will get in French a menu with the following title: Company Profile (English)
This plugin will totally remove the menu item.

== Installation ==

1. Upload `qtranslate-remove-menu-item` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
That's it. It works. No settings needed.
